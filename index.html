Edge Gallery Skills

Custom agent skills for the Google AI Edge Gallery app

How to install a skill

In the AI Edge Gallery app → Agent Skills use case → tap Skills chip → tap (+) → Load skill from URL.

Enter the skill URL, e.g. https://mister-gee.github.io/my-gallery-skills/3d-visualizer/
